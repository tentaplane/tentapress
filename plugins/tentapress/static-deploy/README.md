# Static Deploy

Generate a downloadable static ZIP archive.

## Plugin details

- ID: `tentapress/static-deploy`
- Version: `0.1.2`
- Service provider: `TentaPress\StaticDeploy\StaticDeployServiceProvider`

## Requirements

None.

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/static-deploy`
