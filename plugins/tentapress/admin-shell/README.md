# Admin Shell

Admin UI shell, layout, navigation and base admin routes.

## Plugin details

- ID: `tentapress/admin-shell`
- Version: `0.1.4`
- Service provider: `TentaPress\AdminShell\AdminShellServiceProvider`

## Requirements

None.

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/admin-shell`
