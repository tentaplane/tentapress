# Users

Users, roles, capabilities and admin authentication.

## Plugin details

- ID: `tentapress/users`
- Version: `0.1.4`
- Service provider: `TentaPress\Users\UsersServiceProvider`

## Requirements

None.

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/users`
