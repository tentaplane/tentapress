# Menus

Manage navigation menus and theme locations.

## Plugin details

- ID: `tentapress/menus`
- Version: `0.1.2`
- Service provider: `TentaPress\Menus\MenusServiceProvider`

## Requirements

- `tentapress/users`
- `tentapress/pages`
- `tentapress/posts`
- `tentapress/settings`

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/menus`
