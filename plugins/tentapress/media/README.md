# Media

Upload and manage media files.

## Plugin details

- ID: `tentapress/media`
- Version: `0.1.2`
- Service provider: `TentaPress\Media\MediaServiceProvider`

## Requirements

- `tentapress/users`

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/media`
