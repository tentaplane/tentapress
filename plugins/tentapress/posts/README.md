# Posts

Create and manage blog posts.

## Plugin details

- ID: `tentapress/posts`
- Version: `0.1.3`
- Service provider: `TentaPress\Posts\PostsServiceProvider`

## Requirements

- `tentapress/users`
- `tentapress/blocks`

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/posts`
