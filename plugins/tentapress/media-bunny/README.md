# Media Bunny CDN

Bunny CDN media URL generator.

## Plugin details

- ID: `tentapress/media-bunny`
- Version: `0.1.2`
- Service provider: `TentaPress\MediaBunny\MediaBunnyServiceProvider`

## Requirements

- `tentapress/media`

## Development

- Sync plugins: `php artisan tp:plugins sync`
- Enable plugin: `php artisan tp:plugins enable tentapress/media-bunny`
